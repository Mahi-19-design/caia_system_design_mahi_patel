# 🚀 System Design Concepts API

A simple backend project to manage system design concepts using a clean **MVC architecture**.

---

## 📁 Project Structure

```
├── controllers/
├── models/
├── routes/
├── config/
├── app.js
└── server.js
```

---

## ⚙️ Tech Stack

* 🟢 Node.js
* 🚂 Express.js
* 🍃 MongoDB + Mongoose

---

## 📌 Features

* 📖 Get all concepts
* 🔍 Get single concept
* ➕ Create concept
* ✏️ Update concept
* ❌ Delete concept
* 🗄️ Archive (soft delete using `isArchived`)

---

## 🧠 Concepts Covered

* MVC Architecture
* REST APIs
* MongoDB Schema Design
* Soft Delete (`isArchived`)
* Basic Backend Structure

---

## 🚀 Getting Started

```bash
npm install
npm run dev
```
